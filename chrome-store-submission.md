## 短い説明（128文字以内）

文字列とUnicodeエスケープシーケンスを相互変換できるシンプルなツール。改行を維持、クリックでクリップボードコピー。

---

## 長い説明

## 概要

「Unicode Escape Converter」は、通常のテキストとUnicodeエスケープシーケンス（\uXXXX形式）を相互に変換できるシンプルなChrome拡張機能です。

## 主な機能

✅ **文字列 → Unicode変換**
日本語や特殊文字をUnicodeエスケープシーケンスに変換

✅ **Unicode → 文字列変換**
Unicodeエスケープシーケンスを通常のテキストに戻す

✅ **改行の維持**
複数行のテキストでも改行を維持したまま変換

✅ **ワンクリックコピー**
変換結果をクリップボードに簡単コピー

✅ **プライバシー重視**
完全オフライン動作、データ収集なし

## 使い方

1. Chromeのツールバーから拡張機能のアイコンをクリック
2. 変換方向を選択（文字→Unicode または Unicode→文字）
3. テキストエリアに変換したいテキストを入力
4. 「変換する」ボタンをクリック
5. 「クリップボードにコピー」で結果をコピー

## 変換例

### 文字 → Unicode
```
こんにちは
世界
```
→
```
\u3053\u3093\u306b\u3061\u306f
\u4e16\u754c
```

### Unicode → 文字
```
\u30a2\u30a4\u30a6\u30a8\u30aa
```
→
```
アイウエオ
```

- 個人情報を保存しません

すべての処理はブラウザ内で完結します。
